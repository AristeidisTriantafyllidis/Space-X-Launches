# 🚀 SpaceX Launch Dashboard

A React web application that tracks and displays SpaceX launch data. Users can browse a comprehensive list of missions, view detailed launch statistics, and manage a personalized list of favorite launches.

## ✨ Features
* **Mission Explorer**: Browse a comprehensive list of all SpaceX rocket launches.
* **Advanced Sorting**: Organize launches chronologically by date.
* **Status Filtering**: Filter launches easily by their status (Success / Failure or /UpComing).
* **Favorites System**: Save your favorite missions.
* **Deep-Dive Metrics**: Click into any launch to see full, detailed statistics.

💻 Local Development
Follow these simple steps to get the development environment up and running on your local machine:

Clone the repository:
```bash
git clone https://github.com/AristeidisTriantafyllidis/Space-X-Launches.git
```

Navigate into the project directory:
```bash
cd my-app
```

Install the required dependencies:
```bash
npm install
```

Start the local development server:
```bash
npm start
```
The app will automatically open in your browser at http://localhost:3000.

Run the test suite (Jest):
```bash
npm test
```
